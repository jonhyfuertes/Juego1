# Juego1

Aprendizaje autonomo 1



Generador de Contraseñas (Avance)



1.Descripción del Proyecto



Este proyecto consiste en el desarrollo de un generador de contraseñas utilizando el lenguaje Python.

En esta primera etapa se implementa una versión básica que permite generar una contraseña aleatoria según la longitud ingresada por el usuario.



2.Funcionalidades actuales (Avance)



Solicita la longitud de la contraseña.



Genera una contraseña aleatoria.



Utiliza letras minúsculas.



Implementa estructura repetitiva for.



3.Tecnologías utilizadas



Python 



Librerías:



random



string



4.Estructuras utilizadas



Estructura repetitiva (for)



Variables



Entrada y salida de datos



5.Mejoras pendientes (Próximas versiones)



Agregar opción para incluir mayúsculas.



Incluir números.



Incluir símbolos.



Validar que el usuario ingrese datos correctos.
